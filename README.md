# Portfolio

A personal web portfolio for Shubham Pathak, showcasing software engineering experience and technical projects.

## Setup & Development

```bash
git clone https://github.com/shubham-pathak1/portfolio.git
cd portfolio
npm install
npm run dev
```

## Technical Stack

*   **React 18** with **TypeScript** for strict type-checking and component architecture.
*   **Vite** as the build engine for fast HMR and esbuild-powered transpilation.
*   **Tailwind CSS** for low-level utility-class styling without runtime overhead.
*   **Framer Motion** for spring-physics animations and layout transitions.
*   **React Router DOM v6** for client-side routing.
*   **Lucide React** for consistent SVG iconography.
*   **Lenis** for global smooth scrolling implementation.

## Project Structure

*   `src/components/sections/`: Core page regions (`About`, `Experience`, `Hero`, `Projects`).
*   `src/components/ui/`: Micro-components and interactive UI elements (`Dock`, `ProjectCard`).
*   `src/data/`: Static typed data defining project configurations and professional history.
*   `src/pages/`: Route endpoints (`Home`, `ProjectDetail`).
*   `src/hooks/`: Custom state logic (`useTheme`, `usePerformance`).
